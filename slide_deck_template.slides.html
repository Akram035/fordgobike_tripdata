Investigation Overview¶

Checking the distribution of durations in an appropriate measurement Viewing the number of rides per day and per month Visualizing the ride duration per day and for each user type

Dataset Overview¶

Bay Wheels's trip data From Lyft. The dataset provides information about the bikes rentals in a certain month(I chose the dataset from July 2017) providing columns for rental duration ,start date and time, end station and bikes IDs.

( The distribution of duration values)¶

This plot was made using logarithmic scaleon the x axis. The duration values seem to be normally distributed with some values over 200 making a right tail. It is also clear that durations mostly are between 3 and 30 minutes as most of the points lie between those boundaries. values over 1000 make no sense as it is hard for one rider to ride the bike for a whole day!! and these values are mostley outliers.
